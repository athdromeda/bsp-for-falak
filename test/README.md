# BSP Verification Tests

This directory contains automated tests to verify that the minified BSP kernels generated by `minify.sh` are valid and work correctly with major Python astronomical libraries.

## Prerequisites

- [uv](https://github.com/astral-sh/uv) - Fast Python package manager.
- Generated minified kernels in the `dist/` directory.

## Setup

First, ensure you have generated the minified kernels using the local mode:

```bash
./minify.sh --local
```

Then, install the test dependencies:

```bash
# From the project root
cd test
uv sync
```

## Running Tests

You can run all tests using `pytest` through `uv`:

```bash
uv run pytest
```

This will discover and run all `test_*.py` files in this directory.

### Individual Library Tests

If you want to test a specific library:

- **Skyfield**: `uv run pytest test_skyfield.py`
- **SpiceyPy**: `uv run pytest test_spiceypy.py`
- **Astropy**: `uv run pytest test_astropy.py`

## Test Details

The tests perform the following checks for each `.bsp` file found in `dist/`:

1.  **Loading**: Verifies the library can successfully open and parse the kernel.
2.  **Body Presence**: Ensures the required NAIF IDs (10: Sun, 301: Moon, 399: Earth) are present in the kernel.
3.  **Accuracy**: Performs a sample coordinate calculation (e.g., Moon's position relative to Earth) and checks if the resulting distance is within a reasonable physical range.

## Troubleshooting

- **No kernels found**: Ensure you ran `./minify.sh --local` and that `.bsp` files exist in the `dist/` directory.
- **Leapseconds error**: The `spiceypy` tests require `raw/naif0012.tls` to be present.
